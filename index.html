Conceptos

Style

Style se usa para darle un diseño a nuestra página.

DOM

El DOM es la estructura que representa nuestro código.

Ejemplo

Diagrama en clase

Em y px

Px es una unidad absoluta, se usa para tener tamaños exactos;mientras que em es una unidad relativa al tamaño de la fuente del elemento padre.

Insertar un font

Para insertar un font de google fonts, tienes que hacer click en get font, get embed code, copiar y pegar import url. Por ultimo copiar y pegar el nombre de la fuente.

Divs

Es un contenedor, sirve para agrupar contenido, sin dar un significado especial al contenido.

Ejemplo de div

Nav

Se usa para menus de navegacion, contiene enlaces de navegacion.

Class

Se usa para definir estilos iguales para varios elementos al mismo tiempo.

Sections

Se usa para dividir una pagina web en secciones, tiene un uso mas semantico.

Opciones de menu
